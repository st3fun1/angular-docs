Event Loop

Javascript is single-threaded

JS can only run one task at a time, for long tasks this would be painful, because the UI will be blocked until the task is finished, luckily the browser provides us some features that the JS engine itself doesn't provide: Web API. This inclues DOM API, setTimeout, HTTP request and so on. This help us create some async, non-blocking behavior
