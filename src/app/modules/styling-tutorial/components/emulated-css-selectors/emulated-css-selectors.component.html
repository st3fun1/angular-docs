Emulated CSS Selectors

  • :host

    Targets the host element(the component), which contains the shadow tree

  • :host-context

    Styles some elements based on a parent selector and looks up for it until it reaches the root, it will look for the specified selector on all parent nodes and the actual host itself

  • ::ng-deep combinator

    Shadow Boundary, this combinator allows to pierce through it

    Project content is part of the parent, you need ::ng-deep in order to style it

    !Warning: ::ng-deep unscopes the styling, and it puts it globally, it will apply globally

    ::ng-deep should be used in combination with :host so we scope it to our componen
