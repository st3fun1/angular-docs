Web components

Modular bundle of HTML, CSS and JS which is encapsulated

They are made of:

  • Custom Elements

    Used to create and use our own elements

  • HTML Templates

    Fragments of markup that isn't actually rendered to the screen

    They are used to be referenced and cloned into a specific location

  • Shadow DOM

    Is the encapsulated DOM that we have access to within our components to style and script against

    Cannot be accessed from the light DOM using CSS or JS
